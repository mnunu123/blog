## 배경

**한국도로공사 56주년 영화공모전** — 고속도로와 휴게소를 배경으로 한 56초 자유 형식 영상을 공모하는 대회.

공모전 링크: https://ex-movie.com/

영상 제작 경험이 없는 상태에서 혼자 도전했다. "56주년"이라는 숫자에서 출발해, **개미가 작은 발걸음을 모아 길을 만든다**는 메타포로 스토리를 잡았다. 제목도 그래서 "모여서만든길"이 됐다.

---

## 스토리 구조

영상은 4개의 흐름으로 구성된다.

```
[도입] 개미 클로즈업
  → "작은 발걸음이 모여" (자막)

[과거] 흑백 — 56년 전 고속도로 건설
  → 굴착기, 덤프트럭, 공사 현장
  → 서울↔대전 고속도로 개통식 드라이브 POV

[전환] 컬러 — 고속도로가 사람들의 삶이 된 순간
  → "그렇게, 모두의 여정이 되었다"
  → 차창 밖을 바라보는 사람
  → "그 길은, 서로를 이어주기 위해서라고"

[마무리] 드론 — 현대 고속도로 항공샷
```

개미라는 작은 존재로 시작해서, 56년의 역사를 훑고, 현재의 고속도로로 끝나는 구조다. 흑백→컬러 전환이 과거와 현재를 나누는 경계 역할을 한다.

## 제작 흐름

### Step 1 — 미드저니로 핵심 이미지 생성

GPT로 장면별 프롬프트를 먼저 뽑고, 미드저니에 넣었다. 개미 클로즈업, 흑백 공사 현장, 한국 고속도로 풍경이 이 단계에서 만들어졌다.

![미드저니로 생성한 장면 이미지](/images/ai-video/midjourney-images-1.jpeg)

![미드저니 생성 이미지 2 — 고속도로·인물 장면](/images/ai-video/midjourney-images-2.jpeg)

같은 스타일 프롬프트를 유지해서 화풍의 일관성을 잡았다. 흑백 시퀀스는 미드저니에서 흑백으로 생성하지 않고, 컬러로 먼저 만든 뒤 다빈치리졸브에서 탈색했다.

### Step 2 — GPT로 AI 음악 제작

배경음악도 AI로 만들었다. 고속도로와 한국적 정서를 잇는 방향으로 **한국 민요 스타일**의 AI 음악을 제작해 삽입했다.

![GPT 활용 AI 음악 제작 화면](/images/ai-video/gpt-music-1.jpeg)

![한국 민요 기반 AI 음악 생성 결과](/images/ai-video/gpt-music-2.jpeg)

### Step 3 — 스토리보드 구성

미드저니 이미지들을 56초 타임라인 위에 배치했다. 흑백→컬러 전환 타이밍, 자막이 들어갈 장면, 각 클립의 길이를 미리 계획해두지 않으면 나중에 편집 시 앞뒤가 안 맞는다.

![스토리보드 구성 1](/images/ai-video/storyboard-1.jpeg)

![스토리보드 구성 2](/images/ai-video/storyboard-2.jpeg)

![스토리보드 구성 3](/images/ai-video/storyboard-3.jpeg)

![스토리보드 구성 4](/images/ai-video/storyboard-4.jpeg)

![스토리보드 구성 5](/images/ai-video/storyboard-5.jpeg)

### Step 4 — Kling AI + Veo3로 영상 클립 생성

미드저니 이미지를 **Kling AI**에 넣어 움직이는 클립으로 변환했다. 개미가 실제로 걷고, 굴착기가 움직이고, 차가 달린다. 이미지가 없는 드론샷 같은 보조 장면은 **Veo3**로 텍스트 프롬프트만으로 생성했다.

![Kling AI / Veo3 영상 생성 화면 1](/images/ai-video/kling-veo-generation-1.jpeg)

![Kling AI / Veo3 영상 생성 화면 2](/images/ai-video/kling-veo-generation-2.jpeg)

- **Kling AI**: 이미지를 기반으로 움직임을 생성 → 화풍·색감 일관성 유지에 강함
- **Veo3**: 텍스트 프롬프트만으로 새 장면 생성 → 이미지 없는 보조 장면에 활용

### Step 5 — 다빈치리졸브: 편집 + 자막

클립들을 모아서 56초로 맞췄다. 자막("작은 발걸음이 모여", "56년 전,", "모두의 여정이 되었다")도 이 단계에서 입혔다.

![다빈치리졸브 편집 화면 — 타임라인 구성](/images/ai-video/davinci-edit-1.jpeg)

![다빈치리졸브 자막 작업 화면](/images/ai-video/davinci-edit-2.jpeg)

### Step 6 — 다빈치리졸브: 컬러 그레이딩

가장 공을 들인 단계. Kling 클립, Veo3 클립, 미드저니 원본이 각각 다른 색 톤을 갖고 있어서, RGB를 조절해 전체를 통일했다.

흑백→컬러 전환 효과도 여기서 만들었다. 과거 시퀀스는 채도를 완전히 빼고 대비를 높여서 필름 느낌을 냈고, 컬러 시퀀스와의 경계를 자연스럽게 연결했다.

![다빈치리졸브 컬러 그레이딩 — 흑백 시퀀스 작업](/images/ai-video/davinci-color-1.jpeg)

![다빈치리졸브 컬러 그레이딩 — 컬러 시퀀스 RGB 조정](/images/ai-video/davinci-color-2.jpeg)

## 배운 점

**개미를 메타포로 쓴 게 핵심이었다.** "고속도로 56주년"을 직접 보여주는 대신, 작은 발걸음들이 모여 길이 된다는 이야기로 풀었다. AI로 만든 개미 클로즈업이 실제 다큐멘터리처럼 보여서, 진짜 촬영처럼 오해받기도 했다.

**흑백→컬러 전환이 과거/현재를 나누는 장치가 됐다.** 원래는 단순히 "옛날 느낌"을 내려고 흑백을 썼는데, 컬러로 전환되는 순간이 감정적으로 강한 포인트가 됐다.

**AI 도구마다 잘하는 영역이 다르다.** Kling AI는 기존 이미지를 자연스럽게 움직이는 데 강하고, Veo3는 텍스트만으로 새 장면을 만들 때 낫다. 처음부터 어떤 도구로 어떤 장면을 만들지 역할을 나눠놓지 않으면 화풍이 중간에 달라진다.

**음악을 AI로 만든 것도 유효했다.** 한국 민요 기반 배경음악이 영상의 정서 톤을 잡아주는 역할을 했다.

## 다음 적용 — 코레일 철도건널목 사고 예방 캠페인

같은 파이프라인으로 두 번째 영상을 제작했다.

**프로젝트**: 코레일(한국철도공사) 철도건널목 사고 예방 아이디어 공모전 (고용노동부 연계)
**기간**: 2025.10 ~ 2025.12
**목적**: 철도건널목 안전 수칙을 AI 영상으로 제작해 SNS 배포

첫 번째 영상(고속도로 56주년)에서 검증한 파이프라인을 그대로 재사용했다:
- GPT-4로 안전 수칙 스토리보드 작성
- 미드저니로 핵심 이미지 생성 (철도건널목, 위험 상황 시각화)
- Kling AI로 움직이는 클립 변환
- 다빈치리졸브로 최종 편집 + 자막

**결과**: 도구는 동일, 소재만 달라진 두 번째 사례. 첫 번째에서 구축한 워크플로우가 재사용 가능한 파이프라인임을 확인했다.
